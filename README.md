### ## Operadores binarios lógicos

| Operador | explicación   | ejemplo                  |
| -------- | ------------- | ------------------------ |
| <        | Menor         | 20<30                    |
| >        | Mayor         | 30>40                    |
| <=       | Menor o igual | 20<=30                   |
| >=       | Mayor o igual | 40>=30                   |
| ==       | igualdad      | 20==20   "hola"=="mundo" |
| !=       | no igualdad   | 20!=30  "hola"!="mundo"  |
| &&       | Sí lógico     | n1<n2 &&  n1<n3          |
| \|\|     | Ó lógico      | n1<n2 \|\| 1<n3          |
|          |               |                          |
